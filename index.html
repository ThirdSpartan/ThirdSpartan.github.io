Notes

  • The first dropdown lists all possible item outputs with their type and point cost
  • The next 4 dropdowns represent all possible ingredients with their type and point value
  • Does not support apricorn only/apriball recipes yet
  • Some items do not have a listed value on serebii (labeled UNK) and are treated as adding 0 value
  • "Fixed" recipes require 3 of the same item and the second slot can be anything
  • Will add sorting options for item input lists
  • Thick Club and Luminous Moss listed as a range of values, not sure how to best support that (currently uses smallest value)
